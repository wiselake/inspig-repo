# 월간 리포트 (Monthly Report)

DAY_GB: `MON`

## 1. 개요

월간 리포트는 매월 1일 새벽 3시에 실행되어 지난달의 농장 생산 데이터를 집계합니다.

### 1.1 실행 주기
- **스케줄**: 매월 1일 03:00
- **대상 기간**: 지난달 1일 ~ 말일
- **대상 농장**: TS_INS_SERVICE.REG_TYPE = 'AUTO'인 농장

### 1.2 날짜 계산 예시 (2026-01-01 실행 시)
```
지난달 (리포트 대상): 2025-12-01 ~ 2025-12-31
```
